=======================================
Explicit Blank Areas: EmptySpace Frames
=======================================

Python code file: visual/frame/empty_space.py

`EmptySpace` creates a fixed-size gap in the layout. It can be used for spacing, padding, or layout alignment.

This frame:

- Measures and reserves a rectangular region
- Renders nothing (by design)
- Can be annotated with compatibility fields like `.text` for layout strategies

Useful for layout tuning or placeholder spacing.
